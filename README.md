## Run Locally

Clone the project

```bash
  git clone https://github.com/misaelpaulmamangun/preactexels.git
```

Go to the project directory

```bash
  cd preactexels
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
